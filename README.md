# Digitalbank Landing Page

A responsive implementation of Frontend Mentor's Digitalbank landing-page challenge, featuring a hero, service benefits, article cards, and adaptive navigation.

- **Live demo:** https://digital-banking-snowy.vercel.app
- **Source:** https://github.com/GlitchingghosT/Digital-Banking

## Features

- Responsive desktop and mobile navigation
- Hero and product-benefit sections
- Latest-article cards
- Adaptive call-to-action and footer layout

## Built with

HTML and Tailwind CSS.

## Run locally

No build step is required. Clone the repository and open `index.html` in a browser. Tailwind is loaded through the CDN.
